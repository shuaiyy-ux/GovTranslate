# Chrome Web Store 商店页面内容

## 扩展名称

美政通

## 简短描述

帮助在美华人阅读和理解政府网站内容，提供网页翻译和 AI 术语解释。

## 详细描述（纯文本，直接粘贴到商店）

```
美政通是一款为在美华人设计的浏览器工具，帮助你更轻松地阅读政府网站。

— 网页翻译 —
打开政府网站时，顶部会出现翻译选项，点击即可将页面翻译为中文，保留原始格式。此功能免费，无需配置。

— AI 术语解释 —
选中页面上不理解的文字，点击弹出的解释按钮，AI 会用通俗中文解释含义并给出例子。支持多轮追问。

— PDF 支持 —
打开政府网站上的 PDF 文件时，提供便捷的翻译入口。

— 适用人群 —
留学生、新移民、签证申请者以及在美工作生活的华人。

— 隐私 —
所有数据存储在本地，不收集个人信息。
```

## 分类

生产力工具（Productivity）
