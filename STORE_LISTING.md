# Chrome Web Store 商店页面内容

## 扩展名称

GovTranslate — 美国政府网站智能翻译 + AI 解释

## 简短描述

一键翻译美国政府网站，AI 智能解释移民、税务、签证等术语。支持 Claude / GPT / Gemini / DeepSeek。

## 详细描述

🇺🇸 专为在美华人打造的政府网站翻译工具

在美国办签证、报税、申请移民？看不懂 .gov 网站上的英文？GovTranslate 帮你一键搞定。

━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ 核心功能

【免费网页翻译】
• 打开任意 .gov / .mil 政府网站，一键翻译为中文
• 自动检测政府网站，顶部弹出翻译横幅
• 翻译后的页面保留原始格式和布局

【AI 智能解释】
• 在翻译后的页面上选中任何看不懂的词，点击"💡 AI 解释"
• AI 用通俗中文解释术语含义，并给出具体例子
• 例如：选中"SSN"→ AI 解释"社会安全号码，类似中国身份证号，格式 XXX-XX-XXXX"
• 支持追问，可以继续在气泡中提问

【PDF 翻译】
• 自动检测政府 PDF 文档（如 IRS 税表、USCIS 表格）
• 一键下载并跳转到翻译页面

━━━━━━━━━━━━━━━━━━━━━━━━━━━

🤖 支持四大 AI 模型（自动识别）

粘贴任意一个 API Key，插件自动识别：
• Claude（Anthropic）
• GPT（OpenAI）
• Gemini（Google）
• DeepSeek

网页翻译完全免费，无需 API Key。
AI 解释功能需要 API Key，费用约 ¥0.01/次。

📋 适用场景

• 🎓 留学生：查看学校 .edu.gov 网站、申请 FAFSA 助学金
• 🛂 签证申请：理解 USCIS 网站上的 I-20、I-94、DS-160 等表格说明
• 💰 报税：看懂 IRS 网站的 W-2、1099、1040 等税务术语
• 🏥 医保：理解 Healthcare.gov、Medicare 相关页面
• 🏛️ 其他：SSA 社保、VA 退伍军人事务、各州政府网站

━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔒 隐私保护

• 所有数据存储在本地，不上传到任何服务器
• API Key 仅存储在你的设备上，不会同步
• AI 解释时仅将选中文字发送到你配置的 AI 服务商
• 无广告、无追踪、无数据收集

━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚡ 快速上手

1️⃣ 安装插件
2️⃣ 打开任意 .gov 网站（如 usa.gov、uscis.gov、irs.gov）
3️⃣ 点击顶部蓝色横幅的"翻译"按钮
4️⃣ 在翻译后的页面上选中文字 → 点击"💡 AI 解释"
5️⃣（可选）在插件弹窗中设置 AI API Key 以启用解释功能

💬 常见问题

Q: 翻译需要付费吗？
A: 网页翻译完全免费。AI 解释功能需要 API Key（约 ¥0.01/次）。

Q: 支持哪些网站？
A: 所有 .gov 和 .mil 结尾的美国政府网站。

Q: 繁体中文支持吗？
A: 支持，在设置中切换简体/繁体。

Q: 数据安全吗？
A: 所有数据本地存储，AI 解释仅将选中文字发给你自己配置的 AI 服务商。
